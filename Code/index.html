Base64 编码算法说明

Base64是一种基于64个可打印字符来表示二进制数据的编码方式。

算法原理：

  • 将输入字符串转换为二进制数据（每个字符8位）
  • 将二进制数据按6位一组重新分组
  • 将每组6位二进制数转换为对应的Base64字符
  • 如果最后一组不足6位，则用0填充，并在编码结果后添加相应数量的'='字符

特点：

  • 将任意二进制数据转换为可打印字符（A-Z, a-z, 0-9, +, /）
  • 编码后数据大小增加约33%
  • 常用于在文本协议中传输二进制数据（如电子邮件、网页传输）
  • 不是加密算法（无密钥），但使文本不可直接阅读

示例："Hello" → "SGVsbG8="
